# 🐾 PetAvatar - 宠物3D形象与服装定制平台

## 项目简介

PetAvatar 是一个创新的微信小程序平台，为宠物主人提供两大核心服务：
1. **宠物3D数字形象**：上传宠物照片即可生成3D立体模型，支持小程序内预览和下载为电子桌宠
2. **宠物服装定制**：基于真实体型数据自动生成裁剪图纸，支持在线选款/换布料/定制细节/下单制作

## 技术架构

```
┌──────────────────────────────────────────────────────────┐
│                    C端：微信小程序                         │
│  (原生框架 + Skyline渲染引擎 + threejs-miniprogram)       │
├──────────────────────────────────────────────────────────┤
│                    B端：Web管理后台 (React)               │
├──────────────────────────────────────────────────────────┤
│              后端API服务 (Node.js/Express/TypeScript)     │
│              ├─ 制版引擎 (FreeSewing + 宠物版型公式库)    │
│              ├─ 3D生成服务 (Tripo API → 混元3D)          │
│              ├─ 电商底座 (订单/算价/支付)                 │
│              └─ 积分系统 (签到/兑换/流水)                 │
├──────────────────────────────────────────────────────────┤
│              桌面端：Electron桌宠应用                     │
└──────────────────────────────────────────────────────────┘
```

## 快速开始

### 小程序
1. 使用微信开发者工具打开 `miniprogram/` 目录
2. 配置 `app.json` 中的 appid
3. 安装 threejs-miniprogram 依赖

### 后端
```bash
cd backend
npm install
cp .env.example .env  # 配置环境变量
npm run dev
```

### 数据库
```bash
cd backend
npx prisma migrate dev
```

## 核心功能模块

### 1. 宠物档案
- 基本信息录入（名称/物种/品种/性别/年龄）
- 多角度照片上传（全身/正脸/侧脸）
- 身体尺寸测量（颈围/胸围/背长/腰围）
- 自动体型档位匹配（猫/小型犬/中型犬/大型犬）

### 2. 3D形象生成
- 调用Tripo API从照片生成3D模型
- glTF/glb格式输出
- 自动骨骼绑定 + 基础动画
- 小程序内WebGL实时预览

### 3. 服装定制
- 4大款式：背心/连体衣/四脚套/雨衣
- 16+布料图案：纯色/条纹/碎花/卡通/节日
- 5类细节定制：衣领/花边/袖口/拉链/装饰
- 3D纹理映射实时预览
- 自动算价引擎

### 4. 裁剪图纸生成
- 基于FreeSewing的参数化制版算法
- 自动生成SVG裁剪图纸
- 含裁片轮廓/纱向/缝份/标注
- 支持打印1:1图纸

### 5. 积分系统
- 每日签到（基础10分 + 连续奖励）
- 在线时长积分
- 分享/下单/评价奖励
- 积分兑换虚拟装扮和优惠券

## 技术选型说明

| 技术 | 选择 | 理由 |
|------|------|------|
| 小程序框架 | 原生 + Skyline | 3D渲染性能最优 |
| 3D引擎 | threejs-miniprogram | 官方适配、glTF原生支持 |
| 制版引擎 | FreeSewing (MIT) | 参数化制版、SVG输出、Web友好 |
| 3D生成 | Tripo API → 混元3D | MVP用云API快速验证 |
| 后端 | Node.js + TypeScript | 与FreeSewing同语言 |
| 数据库 | PostgreSQL + Redis | 关系型 + 缓存 |
| 桌宠 | Electron + Three.js | 复用3D资产、透明窗口 |
